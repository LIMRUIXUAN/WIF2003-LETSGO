# WIF2003-EcoTravel-Planner
A full-stack Web Application developed for WIF2003 Web Programming. The Eco-friendly Travel Planner helps users plan sustainable trips by calculating carbon footprints and suggesting green destinations using HTML5, Bootstrap, Node.js, and MongoDB.

# Objective: 
To provide a platform that promotes sustainable tourism through technical solutions like carbon tracking and eco-certified destination filtering.

# Core Features
* User Authentication: Secure login/signup for travelers.
* Carbon Footprint Calculator: Real-time estimation of travel emissions.
* Eco-Destination Discovery: Search and filter for green-certified stays and activities.
* Itinerary Builder: Create and save sustainable travel plans.

# Technology Stack
* Front-End: HTML5, CSS3, JavaScript, and Bootstrap.
* Back-End: Node.js and Express.js.
* Database: MongoDB with Mongoose ODM.
